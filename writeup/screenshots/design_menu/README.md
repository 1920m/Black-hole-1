# Design reference — cinematic main menu

These are reference frames for the planned **cinematic main menu** (see the "Design Direction —
Cinematic Main Menu" section of `writeup/writeup.md`, and Phase 10 in the project specification).

They are stills captured from a short concept clip that styles a black-hole render as a
game-style main menu (an *Interstellar/Gargantua*-inspired look). They are **design inspiration
only** — the menu we build will use our own physically-rendered black hole as the background plate,
not this footage.

| File | What it shows |
|---|---|
| `menu_ref_pov_intro.png` | Opening title card — the black hole framed as a full-screen cinematic plate. |
| `menu_ref_state_newgame.png` | Full menu layout: engraved serif title + tagline, vertical list with square bullets, corner "lore" text. Selection highlight on an upper item. |
| `menu_ref_state_load.png` | Same menu, selection highlight moved down the list (evidences the animated hover state). |
| `menu_ref_state_options.png` | Selection highlight moved again; note the background composition has also shifted (subtle background motion). |

**Provenance:** frames extracted with ffmpeg from a screen-recording supplied by the candidate
(`…/ScreenSketch/.../Recordings/20260626-1333-40.6309874.mp4`, 12.4 s, 1776×1070).
